DORK GENERATOR

Génération de Google Dorks pour l'audit de l'empreinte numérique.

Rappel des opérateurs

  • "…" — expression exacte
  • OR — l'un ou l'autre
  • -mot — exclure un terme
  • site: — limiter à un domaine
  • -site: — exclure un domaine
  • intitle: — dans le titre
  • inurl: — dans l'URL
  • intext: — dans le contenu
  • filetype: — type de fichier
  • * — joker (mot inconnu)
  • ( ) — grouper des termes
  • .. — plage de nombres
  • Une ligne = un dork.

Exporter les dorks de tous les onglets remplis dans une archive ZIP — un fichier .txt par onglet (les onglets vides sont ignorés).
